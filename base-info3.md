### 3. 多线程数据通信的queue.Queue
```
queue.Queue 可以用于多线程之间的、线程安全的数据通信

1. 导入类库
import queue

2. 创建Queue
q = queue.Queue()

3. 添加元素
q.put(item)

4. 获取元素
item = q.get()

5. 查询状态
# 查看元素的多少
q.qsize()
# 判断是否为空
q.empty()
# 判断是否已满
q.full()

```
